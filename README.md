# XD3VQ.github.io

Static GitHub Pages portfolio for Sharan Venkatapathy.

The site is built with plain HTML, CSS, and JavaScript so it can be deployed directly from the repository root. The main page is `index.html`.

## Files

- `index.html` - page content and structure
- `styles.css` - layout, colors, typography, and responsive styles
- `script.js` - mobile navigation, active nav state, and footer year
- `CUSTOMIZATION.md` - notes for updating the portfolio content later

## Local preview

Open `index.html` directly in a browser. No build step or backend is required.
